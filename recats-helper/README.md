## Commands

## Help

```sh
rh help
```

### Github

```sh
# will open github
rh g

# or you may provide a repostiry
rh g owner/repo
```

### Browser

```sh
# will open browser with you search args
rh s "search string"

# or you may provide framework and args
rh s react useEffect
```

### ChatGPT

```sh
# .zshrc -> export OPENAI_API_KEY=""
rh chat "hello, how are you?"
```
